\section{引言}
开发易用且功能强大的三维造型技术一直是计算机图形学的重要目标之一。
专业工具，如Autodesk Maya\textsuperscript{\textregistered}，具有强大的三维造型功能，但操作复杂，需要长时间的学习才能掌握。
那些易用的工具，如Spore Creature Creator\textsuperscript{\textregistered}，将造型过程简化为拼接预定义好的部件，然而造型能力有限。
弥合三维造型工具简单易用与功能强大之间的鸿沟是图形学界最具挑战性的问题之一。

针对该问题，研究人员提出基于草图的造型技术~\cite{LaViola2007}。
这种造型方式简单易用，得到很多关注~\cite{Igarashi1999SG}。
但大多数用户的绘画技术不好，完全按草图很难恢复出具有丰富几何细节的三维模型。
最近几年，研究人员尝试通过草图从三维模型数据库内搜索模型\cite{EitRicBouHilAle12}，或者通过重新组合现有模型的部件产生新模型\cite{Lee2008SBIM}。
这些技术使得普通用户可以快速造型高质量的模型。
然而，这些技术仅允许组合预定义的部件或对部件做简单变形，造型结果单一、功能简单。

在本文内，我们提出一种基于数据驱动的草图式造型方法。
该方法不再局限于使用预定义的部件组合模型，而是实时地从模型数据库内提取出符合造型意图的部件。
我们的方法不需要预分割或标记的三维模型数据库。
我们的技术使用户通过草图简单方便地创建新模型（如图\ref{FigHeadPic}）。
用户仅仅需要具备使用Teddy系统的草图绘制技术，就可以创建出多样的具有丰富几何细节的模型。

实现这样的系统需要解决如下两个技术难题：
1）部件搜索空间非常大。因为模型数据库未预分割，我们不得不实时地找到并切割与用户意图匹配的部件。
为了将搜索空间限制在可解范围内，我们设计了一种平衡策略，使提示的部件在保持可用的前提下尽可能地多样化。
2）我们的方法必须足够快，满足用户交互。
我们发现，如果三维模型的二维投影相似，两三维模型也相似。
因此，我们将三维模型匹配问题转化为多视角下的二维轮廓匹配问题。本文的贡献有三点：
\begin{itemize}
\item 提出一个基于草图的探索式造型方法。使用该方法，用户可以从未预分割的三维模型数据库内，实时提取符合用户意图的个性化部件，创建高质量的模型。
\item 提出一个快速的基于草图的三维模型局部匹配算法。该算法将三维模型的局部匹配问题转化为多视角下的二维轮廓局部匹配问题。这些轮廓组织在一个$k$NN图内。
\item 提出一种新的草图驱动的模型分割算法。
    该算法基于超面片轮廓与候选轮廓间的匹配程度，在模型的超面片图上实时分割出与草图相匹配的部件。
    此外，还有一种新的部件优化算法，进一步优化候选部件。
\end{itemize}
